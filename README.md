# Ting-An Zhou (周庭安) - Portfolio

> 全端軟體工程師 | AI 應用開發 | 敏捷開發實踐者

這是我個人的作品集網站，展示了我的技能、工作經歷、專案作品以及相關證照。網站採用 React.js 開發，並結合 TailwindCSS 打造響應式且具備深色模式美學的現代化介面。

## 🌟 特色

- **星空主題**：沉浸式的動態星空背景與玻璃擬態 (Glassmorphism) UI 設計。
- **響應式設計**：完美支援桌機與行動裝置瀏覽。
- **豐富內容**：
  - 關於我 (About)
  - 核心技能 (Skills)
  - 工作經歷 (Experience)
  - 精選專案 (Projects)
  - 學歷與證照 (Education & Certifications)

## 🛠️ 技術堆疊

- **前端框架**: React.js
- **樣式庫**: TailwindCSS (via CDN for portability)
- **圖標庫**: Lucide React
- **語言**: TypeScript

## 📁 專案結構

- `index.html`: 網站入口點
- `App.tsx`: 主要應用程式邏輯與 UI 組件
- `constants.tsx`: 履歷資料（個人資訊、經歷、專案內容）
- `types.ts`: TypeScript 型別定義

## 🚀 關於我

具備前後端全端開發能力的軟體工程師。曾經使用 Vue.js、React.js 等前端框架以及 ASP.NET、Python Flask、Node.js 進行後端開發經驗。擁有敏捷開發 (Scrum) 實務經驗，致力於結合 AI 技術解決實際問題。
